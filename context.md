# Project Context: Portfolio

This document outlines the technical stack and resources used in the current version of the portfolio website.

## 🛠️ Tech Stack

The project is built using a modern web development stack focused on performance, aesthetics, and smooth user interactions.

- **Framework:** [Next.js](https://nextjs.org/)
- **Library:** [React](https://reactjs.org/)
- **Animations:**
  - [GSAP](https://greensock.com/gsap/) (GreenSock Animation Platform)
  - [Framer Motion](https://www.framer.com/motion/)
- **Smooth Scrolling:** [Lenis](https://github.com/darkroomengineering/lenis)
- **UI Components:** [Skeleton Loader](https://react-loading-skeleton.vercel.app/) (for optimized perceived performance)
- **Styling:** Vanilla CSS / Tailwind CSS (as per project requirements)

## 🎨 Color Theme & Aesthetics

The entire project is built with a **Glassmorphism** aesthetic, providing a consistent, premium, and futuristic feel across all themes.

### 🌓 Theme Toggle
Integrated into the **Navbar**, the toggle switches the **color palette** while maintaining the glassmorphism effects:

- **🌑 Dark Color Theme (Futuristic):**
  - **Concept:** High-contrast, neon-accented, and immersive.
  - **Palette:** Deep space blacks, midnight blues, and vibrant cyan/electric blue accents.
  - **Glassmorphism:** Dark translucent overlays with subtle glow and backdrop-blur.

- **☀️ Light Color Theme (Futuristic):**
  - **Concept:** Ethereal, translucent, and modern.
  - **Palette:** Soft whites, subtle frosted glass textures, and pastel futuristic accents.
  - **Glassmorphism:** Bright, airy translucent backgrounds with heavy backdrop-blur.
